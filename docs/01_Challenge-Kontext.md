# Pacific Dataviz Challenge 2026 – Kontext & Regeln

> Quelle: [pacificdatavizchallenge.org](https://pacificdatavizchallenge.org)
> Veranstalter-Datenplattform: **Pacific Data Hub – `.Stat` Explorer** (Pacific Community / SPC)

## Eckdaten

| Aspekt | Inhalt |
|---|---|
| Thema 2026 | **Climate Change** (Klimawandel) |
| Zeitraum | **1. Juni – 31. August 2026** |
| Preise | über **15.000 USD** Gesamtpreisgeld |
| Teilnahme | kostenlos, jedes Alter, jedes Land/Territorium |
| Tools | frei wählbar (wir nutzen **D3.js**) |
| Einreichung | „Submit your dataviz“ – Dataviz muss **öffentlich** sein |

## Regeln (verbindlich)

1. Die Dataviz **muss mindestens einen Datensatz aus der offiziellen Liste** verwenden.
2. Zusätzliche Datensätze sind erlaubt, sofern sie **offene Daten** sind (z. B. Pacific Data Hub
   oder andere Open-Data-Quellen).
3. Die Dataviz **muss öffentlich zugänglich** gemacht werden.
4. **Alle Datenquellen** müssen im Einreichungsformular gelistet werden.
5. Freie Wahl der Visualisierungstools.
6. Vollständige Regeln liegen als PDF auf der Challenge-Website (Abschnitt *Rules*).

> ⚠️ **Konsequenz für uns:** Mindestens einer der unten gelisteten offiziellen Datensätze muss
> sichtbar in die Visualisierung einfließen. Unser Ozean-Fokus (Meeresspiegel-, SST-,
> Niederschlags-Anomalien) erfüllt dies bereits.
